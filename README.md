# Save as WebP

A Chrome extension that adds a right-click context menu option to save any image as WebP.

## Features

- Right-click any image to download it as `.webp`
- On Google Images, uses the search query as the filename
- Quality set to 75% for a good size/quality balance
- Tiny — less than 100 lines of code

## Installation

1. Clone or download this repo
2. Go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder

## Usage

Right-click any image on a webpage and select **Save image as WebP**.
